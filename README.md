# Tarcin Simulation Template

How to create your simulation:

1. Click the green "Fork" button (top-right)
2. Click on GOLDEN_PROMPT.txt → paste the prompt → click "Commit changes"
3. Open → https://jules.google → click “+ New Project” → paste the forked repo URL

Jules will show you a plan.  
Review the plan → Give suggestions/improvements to the plan if needed → Approve the plan → Jules builds everything → merge the PR → finished!
